# sekai-snake

Многопользовательская 3D-игра "Змейка" с системой кланов, кастомными головами и админ-панелью!

---

## 🛠️ Стек технологий и используемые инструменты

### Backend
- **Node.js** – среда выполнения JavaScript
- **Express** – веб-фреймворк
- **Socket.IO** – библиотека для реального времени
- **Multer** – для обработки файловых загрузок
- **CORS** – для разрешения кросс-доменных запросов

### Frontend
- **Vanilla JavaScript** – без фреймворков
- **Three.js** – 3D-графика
- **HTML5 / CSS3** – разметка и стили

### Хранение данных
- **JSON-файл** (runtime-state.json) – хранение пользователей и серверов в памяти с автосохранением

### Дополнительно
- **Docker** – контейнеризация
- **Render** – хостинг (поддержка через render.yaml)
- **Локализация** – RU/EN
