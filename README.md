# sekai-snake

Многопользовательская 3D-игра "Змейка" с системой кланов, кастомными головами и админ-панелью!

---

## 📋 Описание
Проект представляет собой полноценную многопользовательскую игру на Three.js с серверной частью на Node.js и Socket.IO для реального времени. Игра поддерживает создание кланов, кастомизацию змейки, админ-панель и многое другое!

---

## 🚀 Как начать играть

### 🔧 Системные требования
- Node.js 16+
- npm или yarn

### 📦 Установка и запуск

1. **Склонируйте репозиторий или скачайте проект
2. **Установите зависимости**:
```bash
npm install
```
3. **Запустите сервер**:
   - Для обычный запуск (прод):
     ```bash
     npm start
     ```
   - Для разработки (с автообновлением при изменениях):
     ```bash
     npm run dev
     ```
4. **Откройте браузер** по адресу `http://localhost:3000`

## 🎮 Основные возможности

### Игровые особенности
- **3D-мир** с орфографической камерой
- **Рост змейки** при поедании "нот" (еды)
- **Кланы**: создание, заявки, принятие, исключение, роспуск клана
- **Норки (burrows)**: зона невидимости для игроков
- **Цикл дня/ночи** с плавным изменением цвета поля
- **Лидерборд** в реальном времени
- **Чат** для общения между игроками
- **Джойстик** для мобильных устройств
- **Кастомные головы** вместо стандартной головы змейки
- **Дропы при смерти**: очки выпадают в виде еды

### Лобби и кастомизация
- Авторизация и регистрация пользователей
- Выбор цвета змейки (24 цвета)
- Выбор головы (из админского списка)
- Создание и подключение к серверам (публичные/закрытые с паролем)
- Загрузка собственных голов на модерацию
- Переключение языка (RU/EN)
- Личный кабинет
- Кнопка админ-панели (только для администраторов)

### Админ-панель
- Загрузка и удаление голов для лобби
- Модерация пользовательских предложений голов
- Управление серверами (удаление одного или всех серверов сразу)
- Управление пользователями (удаление)
- Авторизация с проверкой прав администратора

## ⚙️ Настройки и конфигурация

### Основные переменные окружения
- `PORT` — порт для сервера (по умолчанию 3000)
- `ADMIN_USERNAME` — логин админа (по умолчанию `admin`)
- `ADMIN_PASSWORD` — пароль админа (по умолчанию `sekai-admin-041111`)
- `DATA_DIR` — директория для хранения данных (runtime-state.json и ассеты)

### Настройки в коде
- `ADMIN_BONUSES` (server/index-simple.js): бонусы для админов (начальные очки, скорость, неуязвимость)
- `BURROW_SERVER_CONFIG` (server/index-simple.js): настройки норок (количество, размер, время жизни)
- `GRASS_CONFIG` (client/game.js): настройки травы на поле
- `NOTE_TYPES` (client/game.js): типы нот (еды) с разными очками и бонусами
- `GROWTH_CONFIG` (client/game.js): настройки роста змейки и камеры
